# Top100PopCities
World's top 100 most populated cities JSON with latitude and longitude

# How to use it ?

## Install Via npm
```
npm i toppop-cities
```

import the cities array

```
//ES6
import cities from 'toppop-cities'

// NodeJS
const cities = require('toppop-cities');

```
